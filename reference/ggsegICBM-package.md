# ggsegICBM: ICBM Atlas for the 'ggseg' Ecosystem

ICBM white matter atlas for the 'ggseg' ecosystem. Provides a unified
'ggseg_atlas' object with 3D mesh geometry, for use with 'ggseg3d'.
