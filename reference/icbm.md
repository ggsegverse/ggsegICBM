# ICBM White Matter Atlas

Brain atlas for the ICBM DTI-81 white matter labels with 26 tracts.
Contains 3D mesh geometry for
[`ggseg3d::ggseg3d()`](https://lcbc-uio.github.io/ggseg3d//reference/ggseg3d.html).
No 2D polygon geometry is available for this atlas.

## Usage

``` r
icbm()
```

## Value

A
[ggseg.formats::ggseg_atlas](https://ggsegverse.github.io/ggseg.formats/reference/ggseg_atlas.html)
object (subcortical).

## References

Mori et al., MRI Atlas of Human White Matter. Elsevier, Amsterdam, The
Netherlands (2005)
([Elsevier](https://www.elsevier.com/books/mri-atlas-of-human-white-matter/mori/978-0-444-51741-8))
